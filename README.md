# Habit Hole

## Installation & Usage
### Installation
- Clone or download this repo
- Open terminal
### Usage
- Run docker compose up
- If errors, run:   docker-compose down --volumes --remove-orphans
                    docker volume prune --force
- Right click client/index.html and open live server
